Security Information

This application generates cryptographically secure random passwords using the Web Cryptography API. All processing happens in your browser - no passwords are sent to any server.

For maximum security:

  • Use passwords of at least 16 characters
  • Include all character types
  • Use a different password for each service
  • Consider using a password manager
